# Miazo 🎯

**Fast. Clean. Effortless.**

A lightweight desktop overlay application for gamers, streamers, and content creators.

## Features

- ⏱️ **Stopwatch & Countdown Timer** - Track time effortlessly
- 💀 **Death Counter** - Count those unfortunate moments
- 📊 **Progress Bar** - Visual goal tracking
- 📝 **Text Widget** - Display custom text
- 💬 **Chat Feed** - Show live messages
- 🎨 **Glassmorphism UI** - Modern, sleek aesthetic
- ⌨️ **Hotkey Support** - Control everything from your keyboard
- 📱 **Multi-Monitor Ready** - Works across all displays

## Quick Start

```bash
# Install dependencies
npm install

# Run development mode
npm run dev

# Build for production
npm run build:app
```

## Tech Stack

- **Electron** - Cross-platform desktop app
- **React 18** - UI framework
- **Zustand** - State management
- **Vite** - Build tool

## Usage

1. Launch Miazo
2. Press ⚙️ to toggle widgets on/off
3. Use hotkeys to control widgets:
   - `F1` - Toggle Timer
   - `F2` - Increment Deaths
   - `F3` - Reset Deaths

## Customization

All widgets are:
- **Draggable** - Move them anywhere
- **Resizable** - Scale to your needs
- **Themable** - Multiple color schemes
- **Configurable** - Easy JSON config files
